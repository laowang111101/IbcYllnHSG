# 高校宿舍管理系统 java540

## 项目概述

高校宿舍管理系统是基于前后端分离的架构模式，使用Java技术栈进行开发。本项目旨在提供一套高效、稳定、易于维护的解决方案，以帮助高校更好地管理宿舍资源及相关事务。

## 技术栈

- 后端：Spring Boot、MyBatis Plus
- 前端：Vue 3
- 数据库：MySQL
- 开发工具：JDK 8、Maven

## 功能模块

- 用户管理
  - 管理员管理
  - 学生信息管理
- 宿舍管理
  - 宿舍楼管理
  - 宿舍分配管理
- 信息管理
  - 公告发布与维护
- 申请管理
  - 报修申请
  - 调宿申请
- 访客管理
  - 访客预约
  - 访客记录管理

## 系统角色

- 系统管理员
- 宿舍管理员
- 学生用户

## 运行环境

- 服务器：推荐使用Linux操作系统
- 数据库：MySQL 5.7及以上版本
- JDK版本：JDK 8
- 构建工具：Maven 3.x

## 部署步骤

1. 准备好环境，包括Java运行环境和MySQL数据库。
2. 导入数据库脚本，初始化数据库结构。
3. 使用Maven编译后端代码，生成可执行的JAR包。
4. 部署前端代码至Web服务器。
5. 启动后端服务。
6. 通过浏览器访问前端页面。

## 目录结构

```
/高校宿舍管理系统
|-- /backend                后端代码
|-- /frontend               前端代码
    |-- /src
    |-- /public
|-- /doc                    项目文档
|-- /sql                    数据库脚本
```

## 核心亮点

- **高可维护性**：代码结构清晰，采用成熟的框架和工具，易于后续的扩展和维护。
- **灵活的权限控制**：系统针对不同角色的用户设计有不同的权限控制，有效保障系统安全。
